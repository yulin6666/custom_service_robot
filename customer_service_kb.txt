企业内部查询知识库 - 员工常见问题解答

===================
一、行政管理
===================

问：如何申请办公用品？
答：办公用品申请流程如下：
1. 登录企业OA系统，进入"行政管理"模块
2. 点击"办公用品申请"，填写申请单（包括物品名称、数量、用途）
3. 提交后由部门主管审批
4. 审批通过后，行政部门会在3个工作日内配送到您的工作位
常用办公用品（笔、本、订书器等）可以直接到行政前台领取，无需申请。

问：会议室如何预订？
答：会议室预订有两种方式：
1. 在线预订：登录OA系统"会议室预订"模块，选择时间、人数、会议室，提交预订申请
2. 电话预订：拨打行政部分机8888，由行政专员协助预订
预订须知：
- 提前至少2小时预订
- 大型会议室（30人以上）需提前1天预订
- 如需投影仪、白板等设备，请在备注中说明
- 取消预订请提前1小时通知，否则将影响个人信用分

问：公司班车时刻表是什么？
答：公司班车时刻表如下：
【早班车】
- 地铁站A出口：7:30、8:00、8:30
- 地铁站B出口：7:40、8:10、8:40
【晚班车】
- 公司南门：17:30、18:00、18:30、19:00
- 公司北门：17:35、18:05、18:35、19:05
周末和节假日班车时刻表请查看OA系统公告。如遇特殊天气或突发情况，行政部门会提前通知调整。

问：如何申请快递寄送？
答：快递寄送申请流程：
1. 准备好需要寄送的物品和收件人信息
2. 登录OA系统"快递申请"模块填写寄件信息
3. 将物品送至行政部前台，由行政人员统一寄送
4. 快递费用由公司承担（私人物品除外）
私人快递可以寄送到公司，收件地址填写：XX市XX路XX号XX大厦XX楼行政前台，电话：XXX-XXXX。

问：员工工牌丢失怎么办？
答：工牌丢失处理流程：
1. 第一时间报告部门主管和行政部门
2. 登录OA系统填写"工牌补办申请"
3. 到行政部前台拍照，缴纳工本费50元
4. 3个工作日内可领取新工牌
临时出入可凭身份证登记后进入，但需在5个工作日内完成补办。

===================
二、人力资源
===================

问：如何申请年假？
答：年假申请流程：
1. 登录OA系统"请假管理"模块
2. 选择"年假"类型，填写起止日期和事由
3. 提交后由直属主管审批
4. 审批通过后，HR会在系统中扣除相应假期
年假规则：
- 工作满1年享有5天年假，每增加1年增加1天，最多15天
- 年假需提前3天申请，特殊情况可事后补申请
- 年假当年有效，不结转下一年
- 查询剩余年假可在OA"我的假期"模块查看

问：工资发放时间是什么时候？
答：公司工资发放时间为每月15日（遇节假日顺延至下一工作日）。工资构成包括：
- 基本工资
- 岗位津贴
- 绩效奖金（根据上月考核结果）
- 全勤奖（当月无迟到早退）
- 加班费（按实际加班时长计算）
扣除项包括：社保、公积金、个人所得税等。
工资条可在OA系统"薪资查询"模块下载，如有疑问请联系HR部门。

问：社保公积金缴纳比例是多少？
答：社保和公积金缴纳比例如下：
【社保】
- 养老保险：个人8%，公司16%
- 医疗保险：个人2%，公司9.5%
- 失业保险：个人0.5%，公司0.5%
- 工伤保险：个人0%，公司0.5%
- 生育保险：个人0%，公司0.8%
【公积金】
- 个人12%，公司12%
缴费基数为上一年度月平均工资，每年7月调整。查询个人社保公积金账户信息可登录当地社保/公积金官网。

问：如何申请调岗或内部转岗？
答：内部转岗申请流程：
1. 查看OA系统"内部招聘"模块的岗位需求
2. 确认目标岗位后，与现任部门主管沟通
3. 获得主管同意后，填写"内部转岗申请表"
4. 参加目标部门的面试
5. 通过后由HR办理转岗手续
注意：需在现岗位工作满1年才可申请转岗，转岗后试用期为1个月。

问：员工培训如何报名？
答：员工培训报名方式：
1. 关注OA系统"培训中心"的培训公告
2. 选择感兴趣的培训课程，点击"报名"
3. 填写报名信息并提交
4. 培训前1天会收到短信/邮件提醒
培训类型包括：
- 新员工入职培训（必修）
- 专业技能培训（可选）
- 管理能力提升培训（主管及以上）
- 外部培训（需部门主管审批，公司承担费用）

问：离职流程是什么？
答：员工离职流程：
1. 提前30天向直属主管提交书面辞职申请
2. 填写OA系统"离职申请表"
3. 完成工作交接，填写"工作交接清单"
4. 办理离职手续：
   - 归还工牌、办公用品、电脑等公司财物
   - 结清借款、报销等财务事项
   - IT部门注销账号权限
   - HR部门办理离职证明、社保转移等
5. 财务部门结算最后工资（次月15日发放）
试用期内离职需提前3天申请。

===================
三、IT办公
===================

问：忘记OA系统密码怎么办？
答：重置OA系统密码有以下方式：
1. 自助重置：在登录页面点击"忘记密码"，通过绑定的手机号接收验证码重置
2. IT支持：拨打IT服务台分机6666，提供工号和身份验证信息，由技术人员协助重置
3. 现场办理：携带工牌到IT部门办公室办理
为保证账号安全，密码需包含大小写字母、数字和特殊字符，长度不少于8位，且每90天需更换一次。

问：如何申请新的软件或系统权限？
答：软件和系统权限申请流程：
1. 登录OA系统"IT服务"模块
2. 选择"软件/权限申请"
3. 填写申请信息：软件名称、用途说明、使用期限
4. 提交后由部门主管和IT部门审批
5. 审批通过后，IT部门会在2个工作日内完成配置
常用办公软件（Office、浏览器等）已预装，无需申请。专业软件（设计软件、开发工具等）需申请授权。

问：电脑出现故障怎么办？
答：电脑故障处理流程：
1. 简单故障（如软件卡顿）可先尝试重启
2. 无法解决请拨打IT服务台：6666
3. 描述故障现象和紧急程度
4. IT工程师会远程协助或上门处理
故障处理时效：
- 紧急故障（无法工作）：30分钟内响应
- 一般故障：4小时内响应
- 非紧急问题：1个工作日内处理
如需更换硬件（键盘、鼠标、显示器等），请在OA系统提交"硬件更换申请"。

问：如何连接公司VPN？
答：VPN连接步骤：
1. 下载VPN客户端：登录OA系统"IT服务"-"VPN下载"
2. 安装客户端并运行
3. 输入VPN账号（工号）和密码（OA密码）
4. 选择服务器节点，点击"连接"
VPN使用规则：
- 仅用于远程办公和访问内部系统
- 禁止用于下载大文件或P2P下载
- 连接异常请联系IT服务台
- 首次使用需在IT部门进行身份验证

问：公司Wi-Fi密码是什么？
答：公司提供两个Wi-Fi网络：
1. 【内网】Guest-Company（访客网络）
   - 密码：Guest2024（每月更新，请关注OA公告）
   - 仅可访问互联网，无法访问内部系统
2. 【内网】Staff-Secure（员工网络）
   - 需使用工号和OA密码连接
   - 可访问公司内部系统
首次连接员工网络需在IT部门登记设备MAC地址。

问：如何申请企业邮箱？
答：企业邮箱申请：
- 新员工入职后，HR会自动为您开通企业邮箱
- 邮箱格式：姓名拼音@company.com
- 初始密码为：员工工号+Aa123
- 首次登录后请立即修改密码
邮箱登录方式：
1. 网页版：mail.company.com
2. 客户端：可配置Outlook、Foxmail等邮件客户端
3. 手机端：下载企业邮箱APP
邮箱容量为5GB，超出请清理或申请扩容。

===================
四、法务合规
===================

问：如何申请合同审核？
答：合同审核申请流程：
1. 准备合同草稿（Word格式）
2. 登录OA系统"法务服务"-"合同审核"
3. 上传合同文件，填写合同信息（合同类型、金额、对方单位等）
4. 提交后由法务部门审核
5. 法务部门会在3个工作日内反馈审核意见
重大合同（金额超过100万或涉及重要权益）需要7个工作日审核，建议提前规划时间。

问：公司保密协议的主要内容是什么？
答：员工保密协议主要包括：
1. 保密范围：
   - 公司商业秘密（客户名单、价格体系、市场策略等）
   - 技术秘密（技术方案、研发数据、源代码等）
   - 财务数据（财务报表、成本核算、投资计划等）
2. 保密期限：在职期间及离职后2年内
3. 保密义务：
   - 不得泄露给第三方
   - 不得用于个人或其他单位
   - 妥善保管涉密文件和资料
4. 违约责任：违反协议需承担法律责任并赔偿损失
所有员工入职时需签署保密协议，请严格遵守。

问：如何处理知识产权相关问题？
答：知识产权处理原则：
1. 职务成果归公司所有：
   - 员工在职期间完成的发明创造、软件著作、商标设计等归公司所有
   - 公司有权申请专利、著作权登记
2. 申请专利流程：
   - 向法务部门提交"专利申请表"
   - 法务部门进行初步审查
   - 委托专利代理机构办理
3. 著作权登记：
   - 软件、文字作品等可申请著作权登记
   - 向法务部门提交申请和作品材料
如发现侵权行为，请及时报告法务部门处理。

问：员工如何投诉或举报违规行为？
答：投诉举报渠道：
1. 内部投诉：
   - 向直属主管或HR部门反映
   - 拨打投诉热线：8888-1234（匿名可）
   - 发送邮件至：complaint@company.com
2. 严重违规（贪污腐败、商业贿赂等）：
   - 拨打集团监察热线：400-XXX-XXXX
   - 发送邮件至：audit@group.com
3. 匿名举报：
   - 登录公司内网"廉洁举报平台"
公司保护举报人权益，严禁对举报人进行打击报复。经查实的举报，举报人可获得奖励。

===================
五、财务报销
===================

问：如何报销差旅费？
答：差旅费报销流程：
1. 出差前在OA系统提交"出差申请"并获批
2. 出差期间保留相关票据（机票、火车票、酒店发票、餐饮发票等）
3. 出差结束后7天内提交报销：
   - 登录OA系统"费用报销"-"差旅费报销"
   - 上传票据扫描件
   - 填写费用明细（交通、住宿、餐饮、其他）
4. 提交后由部门主管和财务部门审批
5. 审批通过后，款项会在10个工作日内打入工资卡
差旅费标准：
- 交通：经济舱/二等座（总监及以上可选商务舱/一等座）
- 住宿：300元/天（经理级）、500元/天（总监级）、800元/天（VP级）
- 餐饮：100元/天
- 市内交通：实报实销

问：如何报销日常费用？
答：日常费用报销流程：
1. 确认费用类型（办公费、通讯费、交通费、招待费等）
2. 准备发票（必须是公司抬头的正规发票）
3. 登录OA系统"费用报销"-选择对应费用类型
4. 上传发票扫描件，填写费用明细和用途说明
5. 提交审批（金额超过1000元需部门主管审批，超过5000元需财务总监审批）
6. 审批通过后，次月15日随工资发放
注意事项：
- 发票需在开票日期3个月内报销
- 招待费需附客户签到表和会议纪要
- 办公用品需附采购申请单

问：如何查询发票真伪？
答：发票真伪查询方式：
1. 国家税务总局全国增值税发票查验平台：
   - 网址：https://inv-veri.chinatax.gov.cn
   - 输入发票代码、号码、开票日期、金额等信息查询
2. 各地税务局官网或微信公众号
3. 拨打12366纳税服务热线
提交报销前建议先查验发票真伪，假发票不予报销且可能涉及法律责任。

问：公司是否可以代扣个人所得税？
答：是的，公司依法代扣代缴个人所得税。
1. 工资薪金所得：
   - 每月发薪时自动预扣预缴
   - 采用累计预扣法计算
2. 年度汇算清缴：
   - 每年3-6月进行上一年度个税汇算
   - 公司会提供纳税记录和代办服务
   - 员工也可自行通过个人所得税APP办理
3. 专项附加扣除：
   - 子女教育、继续教育、大病医疗、住房贷款利息、住房租金、赡养老人
   - 请在个人所得税APP中填报，或向HR部门提交相关证明
   - 每年12月前更新下一年扣除信息
查询个人纳税记录可登录个人所得税APP或向财务部门申请。

问：如何申请备用金？
答：备用金申请流程：
1. 确定用途（如采购、出差、活动筹备等）
2. 登录OA系统"财务管理"-"备用金申请"
3. 填写申请金额、用途、预计使用时间
4. 提交部门主管和财务部门审批
5. 审批通过后，到财务部领取现金或转账
备用金使用规则：
- 单次申请不超过5000元（特殊情况需总监审批）
- 使用完毕后15天内报销并退还剩余款项
- 逾期未报销将从工资中扣除并影响信用记录

===================
六、采购管理
===================

问：如何发起采购申请？
答：采购申请流程：
1. 确认采购需求（物品名称、规格、数量、用途）
2. 登录OA系统"采购管理"-"采购申请"
3. 填写采购申请单：
   - 物品信息（名称、规格、数量、预算）
   - 需求部门、使用人、到货时间要求
   - 上传技术参数或样品图片（如有）
4. 提交审批：
   - 金额<1万：部门主管审批
   - 金额1-10万：部门总监审批
   - 金额>10万：总经理审批
5. 采购部门会在审批通过后3个工作日内完成采购
紧急采购请在申请单中注明，采购部门会优先处理。

问：采购的供应商如何选择？
答：供应商选择原则：
1. 常规物品：
   - 采购部门有合作供应商名录
   - 优先从名录中选择（质量可靠、价格优惠）
2. 特殊物品：
   - 需求部门可推荐供应商
   - 采购部门会进行比价和资质审核
   - 金额超过5万需至少3家供应商报价
3. 新供应商准入：
   - 提交"供应商准入申请"
   - 提供营业执照、资质证书等材料
   - 采购部门进行评估和现场考察
   - 评估通过后纳入供应商名录
如对供应商有质量投诉，请联系采购部门处理。

问：采购到货后如何验收？
答：采购验收流程：
1. 供应商送货时，收货人需核对：
   - 物品名称、规格、数量是否与采购单一致
   - 外包装是否完好，有无破损
2. 签收后进行开箱验收：
   - 检查物品外观、功能是否正常
   - 核对配件、说明书等是否齐全
3. 验收结果处理：
   - 验收合格：在OA系统"采购验收"模块确认
   - 验收不合格：拍照留证，联系采购部门退换货
4. 登记资产：
   - 固定资产（电脑、家具等）需在行政部门登记
   - 领用物品需签字确认
验收期限为收货后3个工作日，逾期视为验收合格。

问：如何处理采购纠纷？
答：采购纠纷处理流程：
1. 常见纠纷类型：
   - 质量问题（货不对板、有瑕疵）
   - 交货延迟
   - 价格争议
   - 售后服务不到位
2. 处理流程：
   - 第一时间联系采购部门，提供证据（照片、聊天记录等）
   - 采购部门协调供应商处理（退换货、赔偿等）
   - 协商不成，提交法务部门介入
3. 预防措施：
   - 签订采购合同明确质量标准、交货时间、违约责任
   - 付款前务必验收
   - 保留采购凭证和沟通记录
如遇重大纠纷，请及时上报部门主管和采购部门。
